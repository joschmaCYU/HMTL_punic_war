Concise Web Development Summary (HTML, CSS, JS) 1. HTML (HyperText Markup Language) Role: Defines the structure and content of a web page. It's a description language, not a programming language. History: Invented by Tim Berners-Lee (1991), standardized by W3C. Current version: HTML5.
